Website Purpose

This site was created by me to practice and showcase Python coding and web development skills. I developed this project locally, utilizing the Django framework. HTML and CSS are also used for the web pages. All User Posts are fake, and created by me.

The purpose of this site is to be a message board for a college campus. The site is intended to be specifically catered to one school and one school only - hence why the site is VCU themed and the sidebar links are tied to VCU sites. Ideally, if this site was deployed for public use, access would be restricted to students and members of the University, verified by signing up using the students University email. At its core, though, the websites purpose is to demonstrate development skills.
